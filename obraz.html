Język JavaScript został opracowany w 1995 roku
JavaScript jest językiem programowania obsługujący skrypty
Kod JavaScript jest wykonywany przez przeglądarkę
Ze względów bezpieczeństwa JavaScript ma znacznie ograniczone uprawnienia dostępu do zasobów komputera
JavaScript umożliwia lub wymusza stosowanie obiektowych metod programowania.
‹ input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')" ›
"input" stosuje się w obrębie formularzy. Znacznik ten tworzy zarówno jednowierszowe pola tekstowe, pola opcji (checkbox) i pola przełączników (radio button)
"type" Generuje jednowierszowe pole do wprowadzania tekstu.
"name" Atrybut definiuje nazwę elementu
"value" Tekst, który wpisujemy do elementu input , określamy jako wartość (ang. value) tego elementu.
"onclick" Definiuje reakcję na zdarzenie pochodzące od myszy i polegające na kliknięciu.
window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200"); -Standardowa konstrukcja otwierająca okno
toolbar=no -wartość yes lub no (1 lub 0): pokazuje lub ukrywa standardowy pasek narzędzi
directories=no, -wartość yes lub no (1 lub 0): pokazuje lub ukrywa przyciski katalogów
menubar=no, -wartość yes lub no (1 lub 0): pokazuje lub ukrywa menu przeglądarki
height=280, -wartość w pixelach : ustawia wysokość okna
width=160, -wartość w pixelach : ustawia szerokość okna
top=200, -wartość w pixelach : ustawia położenie okna względem góry ekranu
left=200, -wartość w pixelach : ustawia położenie okna względem lewej strony ekranu
window.close() -Standardowa konstrukcja zamykająca okno
